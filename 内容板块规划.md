# 晨间简报Agent - 内容板块规划

> 创建时间: 2026-02-24
> 目标: 每天7:00推送，5分钟读完，把握全天关键信息

---

## 📋 整体架构

```
晨间简报 (每天7:00)
├── 🔥 必读 (3条) - 最重要信息
├── 📊 市场概况 - 一句话总结
├── 🪙 Meme热点 - 新币、大涨、叙事
├── 🎯 Polymarket - 套利机会、新市场
├── 📰 核心资讯 - X/媒体精选
└── 💡 创新雷达 - 新项目、新趋势
```

---

## 板块1: 🔥 必读 (Top 3)

### 设计原则
- 每天只选3条
- 影响力最大的信息
- 必须行动或必须知道

### 筛选标准
```yaml
优先级判断:
  S级:
    - 重大监管政策变化
    - 黑天鹅事件(交易所被黑、战争等)
    - 美联储重大决策
    - BTC/ETH单日涨跌>10%
  A级:
    - 知名项目重大公告
    - 大额鲸鱼操作
    - 重要数据发布(CPI/非农)
  B级:
    - 热门叙事升级
    - 市场情绪极端
```

### 输出格式
```markdown
## 🔥 今日必读 (3条)

### 1. [S级] 美联储暂停加息，BTC突破新高
- 影响: 概率高，立即关注
- 行动: 观察突破后的回踩机会

### 2. [A级] 某知名交易所宣布下架某币
- 影响: 中等，短期波动
- 行动: 持有者注意风险

### 3. [A级] 某meme币24h涨10倍，社区疯狂
- 影响: 情绪指标，谨慎追高
- 行动: 观察不做
```

### 数据来源
- CoinGlass清算数据
- CoinGecko涨跌幅榜
- 监管新闻RSS
- 鲸鱼追踪(WhaleAlert)

## 板块4: 🎯 Polymarket

### 4.1 新市场追踪

```yaml
新市场筛选:
  - 创建时间: < 7天
  - 交易量: > $10k
  - 流动性: 足够
```

### 4.2 套利机会

**高盈利套利** (高风险):
```yaml
条件:
  - 对盘赔率差: > 5%
  - 到期时间: < 7天
  - 流动性检查: 通过
```

**稳定盈利** (低风险):
```yaml
条件:
  - 对盘赔率差: 1-3%
  - 高流动性市场
  - 到期时间: 灵活
```

### 输出格式
```markdown
## 🎯 Polymarket

### 🆕 新市场 (24h)
| 市场 | 是 | 否 | 概率 | 交易量 |
|------|----|----|----|--------|
| BTC3月破10万? | 65% | 38% | | $50k |

### 💰 套利机会

高盈利 (风险):
- 市场A: 是 45% / 市场B: 否 52% → 套利 3.5%

稳定盈利:
- 市场X: 是 49% / 市场Y: 否 50.5% → 套利 0.75%

### 📊 策略监控
- Bitcoin 5m UP/DOWN: 当前信号 [UP] 胜率 55%
- 策略状态: ✅ 运行中
```

### 数据来源
- **Polymarket API**: `https://api.polymarket.com`
- **OpenClaw Webhook**: 策略信号
- **手动计算**: 套利识别脚本
